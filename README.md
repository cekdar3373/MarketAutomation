This project is a C# WinForms-based desktop application developed as a term project.
The application simulates basic market operations such as product management, sales transactions, and stock tracking.

## Project Purpose
The aim of this project is to apply object-oriented programming principles and simulate a real-life market automation system.
Users can add products, manage inventory, perform sales operations, and calculate total prices.

## Features
- Add new products with name, price, and stock information
- List available products
- Add products to cart
- Automatic stock reduction after sales
- Real-time total price calculation
- Complete sales and reset cart

## Used Classes
- Product: Represents a market product with ID, name, price, and stock information

## Technologies
- C#
- Windows Forms (WinForms)
- .NET Framework

## How the Program Works
1. The user adds products through the interface.
2. Products are listed automatically.
3. Selected products are added to the cart.
4. Stock levels decrease automatically.
5. Total price is calculated.
6. The sale is completed and the cart is cleared.

## Project Scope
This project demonstrates a working desktop application that performs real operations instead of only printing outputs.
All operations are handled through class methods and object interactions.
